# Log Analyzer 

A Python-based security tool that analyzes Linux authentication logs to detect failed login attempts, successful logins, and suspicious activities.

---

## Project Goal

Build a lightweight log analysis utility for Linux authentication logs.

---

## Planned Features

- Read Linux log files 
- Detect failed logins
- Detect successful logins
- Extract usernames
- Extract IP addresses
- Generate security reports

---

## Technologies

- Python
- Regular Expressions (Regex)
- File Handling
